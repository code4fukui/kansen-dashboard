# 感染症ダッシュボード

感染症の定点当たり報告数の推移を視覚化したダッシュボードです。

## デモ
[感染症ダッシュボード](https://code4fukui.github.io/kansen-dashboard/)

## 機能
- 今年の定点当たり報告数の推移を表示
- 2013年からの定点当たり報告数の推移を表示

## データ・API
- データソース: [NIID (国立感染症研究所) 速報オープンデータ](https://github.com/code4fukui/niid_go_jp)
- データファイル: [trend.csv](https://code4fukui.github.io/niid_go_jp/data/trend.csv)
